# UI/UX & SEO Rules


- **Design:** Create minimalist, premium, and modern designs.
- **Components:** Follow clean component architecture (shadcn/ui style).
- **States:** Don't neglect loading states (Skeleton screens) and empty states.
- **SEO:** Add a `metadata` object or `generateMetadata` function for every page. Title and description should always include project keywords.
- **Accessibility:** Add `alt` tags to images, use correct HTML semantic tags (`section`, `article`, `header`, etc.).
