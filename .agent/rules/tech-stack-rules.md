# Technology Stack Rules


- **Framework:** Next.js 16 (App Router), React 19. (Use modern React features: `use`, `Server Actions`, etc.)
- **Database/Auth:** Supabase (`@supabase/ssr` and `@supabase/supabase-js`).
- **Styles:** Tailwind CSS (Vanilla).
- **Icons:** Lucide React (`lucide-react`).
- **Charts:** Recharts.
- **Maps:** Leaflet (`react-leaflet`).
